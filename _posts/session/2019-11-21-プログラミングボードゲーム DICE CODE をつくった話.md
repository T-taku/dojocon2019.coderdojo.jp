DICE CODE はプログラミングをモチーフにした対戦型パズルゲームになっています。DICE CODE はどの様にして生まれたのか、これからどこに向かおうとしているのか。企画から制作、販売へと変遷してきたプロジェクトの流れと、これからの目標についてお話させていただきたいと思います。
